Dictionnaire des Cyber-menaces

Comprendre les risques avec des exemples adaptés au contexte sénégalais

Phishing

Moyenne

Faux emails/sites pour voler des mots de passe.

Spear Phishing

Moyenne

Phishing ciblé sur une personne précise (ex. dirigeant).

Whaling Attack

Élevée

Attaque de phishing visant les hauts responsables (PDG).

Malware

Élevée

Logiciels malveillants (virus, trojans, vers).

Ransomware

Élevée

Chiffrement des fichiers contre rançon.

Spyware

Moyenne

Logiciel espion qui surveille tes activités.

Keylogger

Élevée

Enregistre tout ce que tu tapes au clavier.

Rootkit

Élevée

Cache des malwares dans le système sans que l’utilisateur ne voie.

Man-in-the-Middle (MITM)

Élevée

Interception de communication entre deux parties.

Sniffing

Moyenne

Capture de trafic réseau pour voler des infos.

Session Hijacking

Élevée

Vol d’une session active (ex. cookie de connexion).

SQL Injection

Élevée

Insérer du code SQL malveillant dans un site vulnérable.

XSS (Cross-Site Scripting)

Élevée

Injection de scripts dans les pages web.

DNS Spoofing

Moyenne

Rediriger un site vers une fausse adresse.

ARP Spoofing

Moyenne

Usurper une adresse MAC sur un réseau local.

Brute Force Attack

Élevée

Tester plein de mots de passe jusqu’à trouver le bon.

Dictionary Attack

Moyenne

Utiliser une liste de mots de passe courants.

DDoS (Distributed Denial of Service)

Élevée

Saturer un serveur avec trop de requêtes.

Zero-Day Attack

Élevée

Exploitation d’une faille encore inconnue.

Supply Chain Attack

Élevée

Attaquer un fournisseur pour atteindre la vraie cible.
